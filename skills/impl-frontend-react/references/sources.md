## 사용법

### 새 라이브러리 도입 시
1. 위 공식 문서에서 기본 사용법 확인
2. 프로젝트 기존 패턴과 조합 방법 탐색
3. 필요 시 오픈소스 프로젝트에서 실제 사용 예시 참조

### BP에 없는 패턴 필요 시
1. 커뮤니티 가이드에서 검색
2. 공식 문서의 Advanced 섹션 확인
3. 오픈소스에서 유사 구현 찾기
